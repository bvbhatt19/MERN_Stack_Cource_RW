# React Redux Counter (no CSS)

A minimal counter app using React 18, Redux Toolkit, and React-Redux, created with Vite. No CSS styling included.

## Run locally

1. Make sure you have Node.js 18+ installed.
2. Extract the zip.
3. Open a terminal in the project folder and run:
   ```bash
   npm install
   npm run dev
   ```
4. Open the shown local URL in your browser.

## Build for production
```bash
npm run build
npm run preview
```
